TO
SCROLL
RETRIEVE
ALL ITEMS

GET /items

to retrieve the full data set from local database

Content-Type: application/json: JSON data

200 OK Success

It means that the server successfully returned as response all stored items.

500 Internal Server Error

It means that the server encountered an unexpected condition that prevented it from fulfilling the request.

> Go to 1
RETRIEVE
BY ID

GET /items/:id

to retrieve the data of a specific item

Content-Type: application/json: JSON data

200 OK Success

It means that the server successfully returned as response item by ID.

404 Not Found Error

It indicates that the user reached the domain they requested, but the URL path provided no information.

500 Internal Server Error

It means that the server encountered an unexpected condition that prevented it from fulfilling the request.

> Go to 2
CREATE
NEW ITEM

POST /items

to add new item to local database

Content-Type: application/json: JSON data

201 Created

It means that the server successfully created an item.

500 Internal Server Error

It means that the server encountered an unexpected condition that prevented it from fulfilling the request.

> Go to 3
UPDATE
BY ID

PUT /items/:id

to update an existing item by its id

Content-Type: application/json: JSON data

204 No Content

It indicates that the server successfully updates item in local database.

404 Not Found Error

It indicates that the user reached the domain they requested, but the URL path provided no information.

500 Internal Server Error

It means that the server encountered an unexpected condition that prevented it from fulfilling the request.

> Go to 4
DELETE
BY ID

DELETE /items/:id

to delete specific item by id in local database

Content-Type: application/json: JSON data

204 No Content

It means that the server successfully deleted item from database.

404 Not Found Error

It indicates that the user reached the domain they requested, but the URL path provided no information.

500 Internal Server Error

It means that the server encountered an unexpected condition that prevented it from fulfilling the request.

> Go to 5
RESET
TO DEFAULT STATE

GET /reset

to reset local database to its default state, which contains 2 example default items

Content-Type: application/json: JSON data

201 Created Default Items

It means that the database has been reset to its default state, resulting in the creation of two default items.

500 Internal Server Error

It means that the server encountered an unexpected condition that prevented it from fulfilling the request.
